Calling up the unsubscribe form

Calling up the security check confirmation form is done in a mailing. In this mailing, insert a personalized link made up as follows:

http://company.openemm.org/form.do?agnCI=1&agnFN=UnsubscribeQuery&agnUID=##AGNUID##

You must always insert 1 as an agnCI value in OpenEMM. The value for agnFN= is the name of the first form, the one asking the confirmation question, in our example named OptOutQuestion.

Fig. 8.19: A simple HTML form asking the confirmation question from our example.

Please note: Once the recipient has unsubscribed from the mailing list, the link will no longer work, because although the data is still in the database, the system will no longer display it for security reasons. This is why the user ID (agnUID) is no longer valid either, although it features in the personalized unsubscribe link. The administrator interface in OpenEMM will still allow you to call up a user and view his or her details.

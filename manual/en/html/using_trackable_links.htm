Using trackable links

In order to evaluate the success of a mailing, trackable links may be inserted into the text. As soon as a mailing recipient clicks on the link, it calls up a URL linking to the AGNITAS server. The server registers a click and assigns it to a recipient. This works both for text mailings and for HTML mailings. However, the difference is in the notation. The example contains an integrated link. For an explanation, see chapter "Create a link in the text version".

A trackable link measures whether or not a mailing recipient clicked on that link. You can then measure whether or not the recipient was interested in the mailing subject. The chapter on statistics explains about the evaluation of trackable links.

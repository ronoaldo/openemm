Mailing-Liste löschen

Sollten Sie eine Mailing-Liste ganz aus dem System entfernen wollen, rufen Sie wie in Kapitel "Mailing-Liste modifizieren" beschrieben die Übersicht der Mailing-Listen auf. Dann haben Sie zwei Möglichkeiten:

•Klicken Sie in der Übersichtstabelle das Löschen-Symbol hinter der überflüssigen Mailing-Liste an.
•Klicken Sie den Namen der überflüssigen Mailing-Liste an, um den Bearbeitungsdialog aufzurufen. In diesem Dialog klicken Sie unten auf die Schaltfläche Löschen.

Abb. 7.6: Erst wenn Sie die nachfolgende Rückfrage mit der Schaltfläche Löschen bestätigen, entfernt die Software die Mailing-Liste.

In beiden Fällen reagiert der OpenEMM mit einer Rückfrage. Damit ist sichergestellt, dass Sie nicht durch einen falschen Mausklick eine Mailing-Liste verlieren. Erst wenn Sie die Rückfrage mit der Schaltfläche Löschen bestätigen, entfernt die Software die Mailing-Liste aus der Datenbank auf dem OpenEMM-Server.

Bitte beachten Sie: Vor dem Löschen einer Mailing-Liste sollten Sie genau überlegen. Denn eine aus dem Datenbestand gelöschte Mailing-Liste lässt sich nicht ohne weiteres wieder herstellen. Außerdem verlieren Sie mit der Mailing-Liste auch die Permission-Informationen zu den einzelnen Empfängern. Sie können später nicht mehr feststellen, ob ein bestimmter Empfänger jemals die gelöschte Mailing-Liste abonniert hatte oder nicht.
